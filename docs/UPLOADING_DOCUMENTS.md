# Uploading Documents

Supported formats: **PDF**, **PNG**, **JPEG**, **TIFF**

## Process

1. Navigate to **Upload** page
2. Drag & drop or click to select a file
3. The system will:
   - OCR the document (extract text)
   - LLM parses biomarkers and values
   - Normalize results (unit conversion, reference ranges)
   - Store results in your vault
4. View results on the **Results** page

## Supported Lab Reports

Most standard lab reports are supported. The system can handle:
- Lipid panels
- Comprehensive metabolic panels (CMP)
- Complete blood counts (CBC)
- Thyroid panels
- Vitamin & mineral panels
- Hormone panels
- Inflammation markers
- And more (80+ biomarkers)

## Troubleshooting

- **OCR fails**: Try a higher quality scan, or re-export the PDF as text-based
- **Wrong values**: You can manually edit results on the Results page
- **Missing biomarker**: [Add it to the reference data](ADDING_BIOMARKERS.md)
